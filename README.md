# PhoBo · Сайт тестирования поваров (v2)

Двуязычный сайт для проверки знаний ТК поваров ресторана PhoBo. **Русский + Tiếng Việt.**

## 🆕 Что нового в v2

- 🇻🇳 **Вьетнамский язык** — переключатель RU/VI наверху, все тексты переводятся
- 📝 **Аттестация** — повар сам пишет ВСЕ ингредиенты блюда с граммовкой (не выбор из вариантов)
- 🥘 **ПФ разделы** — полуфабрикаты и соусы тоже доступны для тестов и аттестации
- 📊 **Графики прогресса** — видно как растёт балл повара по дням
- ✏️ **Редактирование рецептов** в админке — можешь менять ингредиенты и граммовки
- 📷 **Фотографии блюд** — необязательно, но можно загрузить

## Что получилось

- **`index.html`** — сайт для поваров
- **`admin.html`** — админ-панель для шефа
- **`data.js`** — база из **86 блюд + 45 ПФ** + **979 авто-вопросов** на двух языках

### Сайт повара умеет
- Вход по логину/паролю
- 12 разделов кухни + 3 раздела ПФ
- На каждом разделе **2 режима**:
  - 🔵 **Быстрый тест** (10 вопросов с вариантами ответа)
  - 🟡 **Аттестация** (выбираешь блюдо → пишешь все ингредиенты с граммами, как на бумаге)
- Раздел "Мои результаты" с **графиком прогресса**
- Двуязычный интерфейс (RU/VI), все блюда показываются на двух языках одновременно

### Админ-панель умеет
- **Обзор** — сводка + график динамики кухни
- **Все результаты** — фильтр по повару/разделу/типу/провалам, разбор каждого ответа
- **Аккаунты** — создавай поваров (с именем, логином, паролем). Добавлять может только админ
- **По поварам** — рейтинг кто как справляется + индивидуальные графики прогресса
- **По разделам** — где у кухни слабые места
- **Рецепты** — редактируй ингредиенты, граммовки, добавляй фото блюд. Изменения сразу применяются у поваров

## 🚀 Попробовать прямо сейчас

1. Распакуй `phobo_site.zip`
2. Двойной клик на **`index.html`** → войди как `povar1` / `1234`
3. Открой **`admin.html`** в новой вкладке → войди как `admin` / `admin123`

⚠️ Поменяй пароль `admin123` на свой когда будешь использовать всерьёз!

### Как проверить аттестацию:
1. Войди как повар → нажми «Супы»
2. Выбери "Аттестация" → выбери "Суп Фо Бо"
3. Впиши ингредиенты как помнишь (можно по-вьетнамски, можно по-русски)
4. Нажми "Сдать" → увидишь подробный разбор: что верно, где не та граммовка, что забыл, что лишнее

### Как редактировать рецепты:
1. Зайди в админку → "Рецепты" в меню слева
2. Выбери раздел вверху → кликни блюдо
3. Меняй граммы / добавляй / удаляй ингредиенты / добавляй фото
4. Сохрани — повара сразу увидят новые значения в тестах

## ⚠️ Важно про текущую версию

Сайт работает в **демо-режиме** — данные хранятся в браузере (localStorage). Это значит:

✅ Подходит чтобы пощёлкать и решить готов ли ты к продакшену
❌ **НЕ подходит для реальных поваров на разных телефонах**, потому что:
- Каждое устройство хранит свою копию данных
- Шеф со своего компа НЕ видит результаты повара с его телефона
- Очистка истории браузера = потеря всего

**Чтобы реально использовать с командой — читай `DEPLOYMENT.md`.** Там пошагово как:
- Подключить **Supabase** (бесплатная облачная база)
- Залить сайт на **Vercel** или **Netlify** (бесплатно)
- Привязать к твоему домену

## Как работает аттестация (важно)

**Алгоритм проверки гибкий — не требует точного совпадения текста.** Повар может писать:
- Большими/маленькими буквами
- С опечатками (если не сильные)
- На русском или вьетнамском (мы сравниваем с обеими версиями)
- С приставкой "пф" / "pf" или без неё

**Пример что увидит повар после сдачи:**
- ✅ **Правильно** (зелёным) — ингредиент есть и граммовка верная
- ⚠️ **Не та граммовка** (жёлтым) — ингредиент верный, но число другое (показывает правильное)
- ❌ **Забыл** (красным) — этого ингредиента нет в его ответе, а должен был быть
- ➕ **Лишнее** (серым) — повар написал то, чего в блюде нет

**Балл считается так:** правильные = 1.0, неверная граммовка = 0.5, лишнее = −0.05 за каждое. Делится на общее количество ингредиентов в рецепте.

## База данных

В `data.js` лежит:

- **DISHES** — 86 блюд в 12 разделах: Супы, Горячее, Салаты, Закуски, Бань Бао, Напитки, Кофе, ТК Бабл Ти, Блюда ВК, Курица Депо, Пивные закуски, Утка Сити
- **PF_RECIPES** — 45 ПФ-рецептов в 3 разделах: ПФ (начинки/заготовки), ПФ Соус, ПФ Бабл Ти
- **QUESTIONS_DB** — 979 авто-сгенерированных вопросов на двух языках

Все блюда автоматически вытащены из твоего файла `ТК_PhoBo_20_04_26.xlsx`. Если что-то парсилось криво — поправь в админке через "Рецепты".

## Если что-то не так

- Не входит как админ? Открой консоль (F12) → `localStorage.clear()` → обнови страницу. Логин будет `admin`/`admin123`
- Хочешь сбросить все рецепты к исходным? Та же команда `localStorage.clear()`
- Сломалось после редактирования? Очистка `localStorage.removeItem('phobo_dish_overrides')`

Если что-то непонятно или не работает как ожидаешь — пиши, поправлю.
